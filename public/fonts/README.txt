Yerel font dosyalarınızı buraya koyun:
- sans.woff2 (normal, 400)
- sans-bold.woff2 (kalın, 700)

Sadece normal ağırlık kullanacaksanız sadece sans.woff2 yeterli; sans-bold.woff2 yoksa kalın metin sistem fontuna düşer.

Farklı dosya adı kullanacaksanız app/globals.css içindeki @font-face src yolunu güncelleyin.
